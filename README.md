# Solid Examples (Learning Repo)

## English

This repository contains my study notes and small C# examples created to learn SOLID principles.  
It is not intended for production use; it is for learning and practice.

### Purpose
- Understand SOLID principles with concrete examples
- Practice refactoring from poor design to better design
- Build a strong foundation for applying SOLID in larger real-world projects

### Scope
- Single Responsibility Principle (SRP)
- Open/Closed Principle (OCP)
- Liskov Substitution Principle (LSP)
- Interface Segregation Principle (ISP)
- Dependency Inversion Principle (DIP)

### Note
This repository is a set of study notes. Some parts may be incomplete or open to improvement.  
Feedback and contributions via Issues/PRs are welcome.
